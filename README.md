# Spotify Podcast Description Enhancer

## 📝 プロジェクト概要

Spotifyのポッドキャスト概要欄を、より使いやすく、見やすい形式に変換するWebアプリケーションです。

### 🎯 解決する課題

現在のSpotifyポッドキャストの概要欄には以下のような問題があります：

- リンクの表示が分かりにくく、クリックしづらい
- 長文テキストの可読性が低い
- タイムスタンプへの直接ジャンプが困難
- 関連リソースへのアクセスが不便

### 🚀 主な機能

1. **スマートリンク変換**
   - 埋め込まれたURLを見やすいカード形式に変換
   - サムネイル画像とタイトルの自動取得

2. **タイムスタンプナビゲーション**
   - ポッドキャスト内の各セクションへ簡単にジャンプ
   - 視覚的なタイムライン表示

3. **コンテンツ構造化**
   - 箇条書きや見出しの自動認識
   - 読みやすい階層構造への変換

## 🛠 技術スタック

- **フロントエンド**: React + TypeScript
- **バックエンド**: Hono
- **認証**: Spotify OAuth
- **データベース**: Cloudflare D1
- **ホスティング**: Cloudflare Pages
